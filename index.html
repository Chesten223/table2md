Frequently Asked Questions

How do I convert an HTML table to Markdown?

Paste your HTML table code into the input area. The converter automatically detects the table and outputs valid Markdown table syntax. You can also drag and drop an .html file onto the textarea.

Does the tool support merged cells (colspan/rowspan)?

Yes. Merged columns (colspan) are expanded by repeating the cell content. Merged rows (rowspan) are filled in subsequent rows. Markdown does not natively support spanning, so the content is duplicated to preserve data integrity.

Can I convert CSV or TSV to a Markdown table?

Yes. Select the CSV or TSV input mode, paste your data, and the first row is treated as the header. The tool handles quoted fields and tab-separated values automatically.
